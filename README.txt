Software Engineering: Project Management Application Project

Team Members: Cameron Boydd, Bryant De Luna-Peralta, John Caraway, and Braxton Haight

This project demonstrates Flask, Python, HTML/CSS, Javascript and SQLAlchemy

The features of this application include, but are not limited to:
Sign in/ Register/ Sign out, Password validation, Create a project, alphabetize project list, 'favorite' action to list projects in priority, add or delete a task for project, add or delete comments and upvote or downvote and list them in that priority, share projects between user accounts
